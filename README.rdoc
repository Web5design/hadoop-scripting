== downloading some sample data

run only once:

  sudo gem install curb sequel
  sudo gem install do_mysql -- --with-mysql-dir="/usr/local/mysql/" --with-mysql-config="/usr/local/mysql/bin/mysql_config"
  git submodule init
		

run to download some wikipedia articles:
  
  rake get_data:wikipedia
		
download 10 MB to data/foo

  rake get_data:wikipedia MB=10 FILE=data/foo
